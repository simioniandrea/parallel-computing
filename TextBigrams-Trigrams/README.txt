Per eseguire da Terminale:
	-Sequential
		1) Recarsi all'interno della directory Sequential
		2) $ javac Main.java -d out
		3) Con JRE Error $ javac -target 8 -source 8 Main.java -d out 
		4) $ cd ./out
		5) $ java Main
		6) Ricordarsi di cambiare i path inizialmente all'interno di Main.java

	-Parallel
		1) Recarsi all'interno della directory Parallel
		2) $ javac Main.java -d out
		3) Con JRE Error $ javac -target 8 -source 8 Main.java -d out 
		4) $ cd ./out
		5) $ java Main
		6) Ricordarsi di cambiare i path inizialmente all'interno di Main.java
----------------------------------------------------------
Per eseguire con IntelliJ
	-Sequential: Aprire il progetto Sequential con ItelliJ
	-Parallel: Aprire il progetto Parallel con ItelliJ
